Introduction

Python is a high-level, interpreted, and versatile programming language. It was created with an emphasis on code readability and simplicity, making it an excellent choice for beginners and experienced developers alike.

Python comes with a comprehensive standard library, providing a wide range of modules and packages for various tasks, such as file handling, networking, and data manipulation. It also offers a core set of language features, including data types, control structures, and functions, that form the foundation for building powerful applications.

  • General Python extends the core language and offers a plethora of modules and libraries to accomplish a wide array of tasks. For example, you can use modules to handle regular expressions, perform mathematical computations, and work with dates and times.
  • Web Development with Python provides frameworks like Django and Flask, which facilitate the development of web applications, handling HTTP requests, and generating dynamic content.
  • Data Science with Python involves using libraries like NumPy, Pandas, and Matplotlib for data manipulation, analysis, and visualization.
  • Scripting with Python allows you to automate repetitive tasks, interact with files, and manage system resources efficiently.
  • Machine Learning and Artificial Intelligence can be achieved using libraries such as TensorFlow and PyTorch, empowering developers to build intelligent applications and models.
  • Server-side Python, with frameworks like Flask and FastAPI, enables the development of robust APIs and web services, handling client requests and serving dynamic content.
What you should already know

This guide assumes you have the following basic background:

  • A general understanding of computer programming concepts and principles.
  • Good working knowledge of Python programming language.
  • Some familiarity with basic programming constructs such as variables, loops, and functions.
Hello world
To get started with writing Python, open your preferred Python environment or code editor and write your first "Hello world" Python code: print("Hello world") Run the code, and you should see the "Hello world" message displayed in the output.
Variables
In Python, you use variables as symbolic names for values in your application. Variable names, also known as identifiers, must conform to certain rules:
  • A variable name must start with a letter (a-z, A-Z) or an underscore (_).
  • Subsequent characters in the variable name can be letters, digits (0-9), or underscores.
  • Variable names are case-sensitive, meaning "myVariable" and "myvariable" are considered different variables.
You can create and assign values to variables using the assignment operator (=). For example: age = 25 Python is a dynamically typed language, meaning you do not need to explicitly specify the variable type. Python will infer the type based on the assigned value.
Data types
Python supports several data types, including:
  • int: Integer values, e.g., 42, -10, 0.
  • float: Floating-point values, e.g., 3.14, -0.5.
  • str: String values, e.g., "Hello", 'Python'.
  • bool: Boolean values, either True or False.
  • list: An ordered collection of elements, e.g., [1, 2, 3].
  • tuple: Similar to lists but immutable, e.g., (1, 2, 3).
  • dict: A collection of key-value pairs, e.g., {'name': 'John', 'age': 30}.
  • set: An unordered collection of unique elements, e.g., {1, 2, 3}.
Python uses dynamic typing, allowing variables to change data types during runtime. For example, you can assign an integer value to a variable and later reassign it to a string value.
Conditional statements
Python uses if...elif...else statements for conditional execution. The basic syntax is as follows: if condition: # code to execute if the condition is True elif another_condition: # code to execute if another_condition is True else: # code to execute if none of the above conditions are True Python evaluates the conditions one by one and executes the code block associated with the first True condition encountered. If none of the conditions are True, the code block under else (if present) will be executed. Note that the elif (short for else if) part is optional, and you can have multiple elif blocks if needed.
Loops
Python supports two primary types of loops: for loop and while loop. The for loop is used to iterate over a sequence (e.g., list, tuple, string) and execute a block of code for each element in the sequence. The basic syntax is as follows: for item in sequence: # code to execute for each item in the sequence The while loop is used to repeatedly execute a block of code as long as a given condition is True. The basic syntax is as follows: while condition: # code to execute as long as the condition is True Be cautious when using a while loop to ensure the loop eventually terminates to prevent infinite loops.
Functions
Functions are reusable blocks of code in Python. They help organize code and promote code reusability. You can define a function using the def keyword, followed by the function name and a set of parentheses containing any function parameters. def greet(name): return "Hello, " + name The above function, greet(), takes one parameter called "name" and returns a greeting message using that name. You can call the function by passing an argument: print(greet("John")) # Output: "Hello, John" Functions can have return statements that specify the value the function will return when called. If a function does not have a return statement, it implicitly returns None.
Lists
Lists are one of the most commonly used data structures in Python. They are ordered collections of elements and are mutable, meaning you can change their contents after creation. To create a list, use square brackets [] and separate the elements with commas. fruits = ['apple', 'banana', 'orange'] You can access individual elements in a list using indexing: print(fruits[0]) # Output: 'apple' print(fruits[1]) # Output: 'banana' Python also provides various list methods for manipulation, such as append(), remove(), pop(), and more.
Dictionaries
Dictionaries are collections of key-value pairs. Each value in a dictionary is associated with a unique key, allowing efficient data retrieval. To create a dictionary, use curly braces {} and separate the key-value pairs with colons. person = {'name': 'John', 'age': 30, 'city': 'New York'} You can access the values using the keys: print(person['name']) # Output: 'John' print(person['age']) # Output: 30 Dictionaries are also mutable, so you can add, modify, or delete key-value pairs as needed.
Classes and objects
Python is an object-oriented programming language. It supports classes and objects for building reusable and modular code. A class is a blueprint for creating objects, and an object is an instance of a class. class Dog: def __init__(self, name, age): self.name = name self.age = age def bark(self): return "Woof!" my_dog = Dog("Buddy", 3) print(my_dog.name) # Output: "Buddy" print(my_dog.age) # Output: 3 print(my_dog.bark()) # Output: "Woof!" In the above example, we define a Dog class with attributes (name and age) and a method (bark). We create an instance of the Dog class and access its attributes and methods using dot notation.
